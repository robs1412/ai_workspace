# Taster Reminder Product-Carry Note

Use this in Vanessa / National Outreach staff reminders when account product history is available.

Source priority:

- Existing OPS event notes if they already name the products.
- Salesreport Chain Store Intelligence / Chain Invoice Report for chain accounts and ordinary retail tastings.
- Recent Binny's scraper output for Binny's current-placement context when needed.

Reminder line:

`Account history suggests they carry: [products]. Please bring samples/materials for: [products or priority focus].`

If account history is missing:

`Product history is not confirmed in Salesreport; bring the standard tasting kit plus any products named in the OPS event notes.`

Do not overbuild this into a scraper-vs-invoice analysis for ordinary reminders. The goal is practical sample planning for the person working the tasting.
